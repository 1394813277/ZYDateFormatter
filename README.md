# ZYDateFormatter
This extension is used to solve the problem of NSDateFormatter performance

###useage

``` Objective C
NSDate *date = [NSDate date];
NSString *resultStr = [date stringWithCStyleFormatter:@"%G - %m - %d %H:%M:%S"];
```
